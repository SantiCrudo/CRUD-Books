# CRUD de Libros

Proyecto backend desarrollado con **Node.js**, **Express** y **MySQL** para gestionar una lista de libros.  
Incluye operaciones de **crear, leer, actualizar y eliminar (CRUD)** sobre una tabla `libros`.

---

## Tecnologías utilizadas

- Node.js
- Express
- MySQL
- mysql2
- dotenv
